Printing

In order to start a java program, there are a couple things to watch for, first is where you start your program. In most IDEs (Integrated development environment) you will see a couple different headers one may include 'public class' which is followed by the name of your file. You will want to look for a header that is labled: 'public static void main(String[] args)' which is where you will want to leave most of your code, atleast for the time being. You should see an opening and closing braces('{}') you should put your code between these two braces in order for the code to properly execute.

Printing in Java can be done by a single line, yet there are different types of outputs that the code can execute
